# Getting Started

## Prerequisites

- Node.js v20+
- npm

## Setup

```bash
npm install
```

## Development

```bash
npm run dev
```

Opens at [http://localhost:3000](http://localhost:3000).

## Production Build

```bash
npm run build
npm start
```

## Linting

```bash
npm run lint
```

## Tech Stack

- Next.js 16 (App Router, Turbopack)
- React 19
- TypeScript
- Tailwind CSS 4
